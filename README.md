# CP-You2
シンプルなCPU使用率表示ツール

## Overview
CPU使用率表示ツール  
平均値、スレッド毎の使用率を表示することができます。  
HUDライクな見た目で、デスクトップを占有しません。  

## QuckStart
CP-You2.exe を起動するだけ。  
ドラッグで好きな場所に移動したり、右クリックでモードを切り替えたり、タスクトレイアイコンから終了や常に前画面に表示するモードすることができます。
